# 示例壁纸

本目录描述 Wall 便携包中的三个测试视频：

- `01-landscape-1280x720.mp4`
- `02-portrait-720x1280.mp4`
- `03-square-720x720.mp4`

这些文件由 `scripts/create-sample-wallpapers.ps1` 使用 FFmpeg 的 `testsrc2` 测试源生成，不包含第三方
照片、视频、音乐、字体或商标素材。生成结果随 Wall 一同采用项目根目录中的 MIT License。

示例仅用于验证导入、循环播放和 Cover/Contain/Stretch 缩放，不代表正式壁纸内容。
