# 关于页作者信息设计

## 目标

在“设置 → 关于”页面展示项目作者和联系邮箱，便于开源用户识别维护者并反馈问题。

## 界面

- 在产品说明“本地视频与图片动态壁纸工具”下方、操作按钮上方增加两行文本。
- 文案固定为“作者：NiceBlueChai”和“联系邮箱：bluechai@qq.com”。
- 沿用现有次级文字颜色和字号，不新增图标、按钮、链接或弹窗。

## 行为与边界

- 信息仅静态展示，不发起网络请求，不调用系统邮件客户端。
- 不新增配置项、状态、Rust 命令或依赖。
- 适当减少关于面板现有间距，确保固定高度内不溢出。

## 验证

- Vue 组件测试断言作者和邮箱在“关于”路由中可见。
- 执行完整前端测试和生产构建。
