# 分类编辑交互实现报告

日期：2026-07-21

## 用户契约

用户可以：

- 在详情信息卡中查看当前分类，并直接把壁纸从某个分类移出。
- 通过“添加分类”菜单把壁纸加入已有分类；已经加入的分类显示勾选且不可重复提交。
- 在没有任何分类时，从详情页或批量管理中直接“新建并添加”，不需要先离开当前任务。
- 在批量管理中把所选壁纸加入已有分类，或创建分类并自动添加全部所选壁纸。
- 在分类创建后、添加失败时看到明确的部分成功提示，并保留已经创建的分类。

用户不能：

- 重复把同一壁纸添加到已经拥有的分类。
- 在没有分类时打开“从分类移除”菜单。
- 通过分类编辑删除、移动或修改任何壁纸源文件。
- 在异步分类操作完成前重复提交同一操作。

## 失败与恢复

- 分类名称为空或超过 40 个 Unicode 字符时留在弹窗内就地提示。
- 创建失败时保留弹窗和输入，恢复输入焦点。
- 创建成功但成员关系写入失败时关闭弹窗、恢复触发点焦点，并提示“分类已创建，但添加到当前壁纸失败，请重试”。
- 详情或批量成员关系写入失败时保留当前页面状态并显示错误，不伪造成功状态。
- 菜单和弹窗支持方向键、Home、End、Esc、焦点返回和弹窗焦点陷阱。
