# 版本记录

本项目遵循语义化版本。发布日期使用 `YYYY-MM-DD`。

## [Unreleased]

### Added

- 增加用户分类、分类管理和壁纸批量增删分类。
- 增加单项和批量“从库中移除”及失效项全量扫描清理，所有操作均保留用户源文件。
- 增加详情页和批量管理的零分类“新建并添加”流程。
- 增加独立、复制、铺展三种多显示器模式，以及按显示目标操作的托盘子菜单。
- 增加常用画幅、三级抗锯齿、源/24/30/60 FPS 和壁纸级设置覆盖。
- 增加最大化应用暂停、显示器热插拔保留和复制组 100 ms 漂移校准。
- 增加无需鼠标的便携版原生冒烟测试和中断恢复流程。

### Changed

- 详情页恢复应用内视频预览，图片隐藏视频专属控制，并优化长文件名、完整路径和分类编辑布局。
- 关于页增加作者与联系邮箱，启用 GitHub 项目主页，并与 Figma 设计保持一致。
- 补充开源协作、安全、产品契约和持续集成文档。
- 便携包改用项目脚本生成的测试视频，不再分发来源不明的媒体。
- 便携包增加三张与 2560 × 1440 主屏匹配的原创图片测试壁纸。
- 会话恢复会重建全部显示目标，并恢复用户手动暂停与各目标静音状态。
- 同一显示目标上的兼容视频切换改为通过 mpv IPC 热加载，不再重复创建播放器进程。

### Fixed

- 修复恢复上次壁纸时，后台监视器与 WebView 主线程锁反转导致的启动未响应。
- 修复自动暂停提交与前台切换之间的播放器/核心状态锁反转，避免切换时长时间无响应。
- 修复状态提交门闩覆盖界面事件与托盘刷新时可能产生的竞态死锁；通知改由单一后台队列顺序发布。
- 修复 mpv IPC 响应读取器写死请求 ID，导致热切换等待超时后错误重启播放器。
- 修复热切换回滚不完整时仍保留损坏播放器组，避免核心状态与桌面实际媒体不一致。
- 加固便携验证的中断恢复流程，避免运行中的 Wall 或残留备份覆盖用户应用数据。
- 修复开机启动关闭时，修改播放设置会因删除不存在的系统启动项而失败。
- 修复图片壁纸错误应用视频 FPS 滤镜后导致 mpv 退出、桌面无法持续显示图片。

## [1.0.0] - 2026-07-16

### Added

- 本地视频和图片壁纸库、详情、搜索和筛选。
- WorkerW 桌面嵌入与 mpv 播放控制。
- Cover、Contain、Stretch、音量、静音、暂停和停止。
- 全屏、电池和显示器休眠自动暂停。
- 托盘、单实例、开机启动、关闭到托盘和会话恢复。
- Windows 10/11 x64 完全离线便携包。
